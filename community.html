Community Guidelines

Code of Conduct

We are committed to providing a welcoming and inclusive environment for all community members. Our community is built on respect, collaboration, and mutual support.

Our Values

  • Be Respectful: Treat everyone with kindness and respect, regardless of their background or experience level.
  • Be Helpful: Share your knowledge and help others learn and grow.
  • Be Open: Welcome new ideas and different perspectives.
  • Be Professional: Maintain a professional and constructive tone in all interactions.

Reporting Issues

If you encounter any behavior that violates our community guidelines, please report it to the maintainers through our contact page or via email.

Contributing

How to Contribute

We welcome contributions from the community! Whether it's fixing bugs, adding features, improving documentation, or helping others, every contribution makes a difference.

Ways to Contribute

  • Code Contributions: Submit pull requests for bug fixes, new features, or improvements.
  • Documentation: Help improve our documentation, fix typos, or add examples.
  • Bug Reports: Report bugs and issues you encounter while using CoreArenaCLI.
  • Feature Requests: Suggest new features or improvements.
  • Community Support: Help answer questions in discussions and forums.
  • Testing: Test new releases and provide feedback.

Getting Started

  1. Fork the repository
  2. Create a branch for your contribution
  3. Make your changes
  4. Submit a pull request with a clear description

For more detailed contribution guidelines, please check our CONTRIBUTING.md file.
