Core Components

CoreArenaCLI consists of three integrated components working together

{% include icon.html name="cli" %}

codearena-cli

The command-line interface built with oclif that provides a powerful CLI for interacting with the CoreArena platform.

  • Modular command structure
  • Extensible architecture
  • Cross-platform support
  • Developer-friendly API
{% include icon.html name="server" %}

codearena-server

The backend REST API built with Express.js that handles persistent data, authentication, and code execution.

  • RESTful API design
  • Database models & migrations
  • OAuth authentication
  • Docker sandbox integration
{% include icon.html name="web" %}

codearena-web

A modern Next.js web application providing a user-friendly interface for the CoreArena platform.

  • React-based UI
  • Server-side rendering
  • Responsive design
  • Real-time updates
